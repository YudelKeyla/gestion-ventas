📖 Manual de Usuario

📦 Productos

Aquí agregas, editas y eliminas los productos que vendes.

  • Agregar: Completa el formulario con nombre, precio de costo, precio de venta, stock inicial y fecha de compra. Si el producto ya existe, se suma al stock.
  • Editar: Toca el ícono ✏️ en cualquier producto para modificar sus datos.
  • Buscar: Usa la barra de búsqueda para encontrar productos rápidamente.
  • Exportar: Con el botón 📤 descargas la lista en formato Excel (.tsv).
  • Eliminar todos: El botón 🗑️ borra toda la lista (pide confirmación).

💰 Vender

Registra una venta de forma rápida.

  • Seleccionar producto: Toca el campo "Toca para seleccionar..." y elige el producto del catálogo. Puedes buscar dentro del modal que aparece.
  • Cantidad: Indica cuántas unidades vendes.
  • Carrito: Agrega productos al carrito. Puedes quitar items o vaciarlo.
  • Finalizar venta: Al confirmar, se descuenta el stock automáticamente y se guarda en el historial.

📋 Historial

Consulta todas las ventas realizadas.

  • Filtra por fecha específica o por mes/año.
  • Visualiza el total de ventas y ganancias del día y del mes.
  • Exporta el historial completo con el botón 📤.
  • Limpia todo el historial con el botón 🗑️.

📊 Estadísticas

Obtén un resumen visual de tu negocio.

  • Revisa indicadores como ventas totales, promedio, mejor día y unidades vendidas.
  • Gráfico de barras: ventas de los últimos 7 días.
  • Gráfico circular: productos más vendidos.
  • Exporta un reporte completo en formato Excel.

💾 Respaldo y Restauración

Protege tu información.

  • Descargar Respaldo: Guarda un archivo .json con todos tus datos (productos e historial).
  • Cargar Respaldo: Restaura los datos desde un archivo .json previamente guardado. Ideal para migrar a otro dispositivo o recuperar tras una actualización.

🔐 Seguridad

La app se protege con un PIN de 4 dígitos (por defecto: 1234). Puedes cambiarlo en el código fuente.

📴 Modo Offline

La aplicación funciona completamente sin internet. Los datos se guardan en la memoria del dispositivo. Solo necesitas conexión la primera vez para cargar los gráficos.
